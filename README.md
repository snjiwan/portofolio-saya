# Portofolio - AKSAN KURAJI DERMAWAN

Fullstack Web Developer | Front End Developer

Portfolio website interaktif dengan animasi Three.js, upload CV & penghargaan.

## Fitur

- Particle 3D background (Three.js)
- Typing effect (Typed.js)
- Scroll animations (AOS)
- Custom cursor glow & magnetic hover
- Dark theme glassmorphism design
- Upload CV (PDF/DOC)
- Upload foto profil
- Upload & manage penghargaan
- Admin panel
- Fully responsive

## Teknologi

- HTML5, CSS3, JavaScript
- Three.js, Typed.js, AOS
- PHP (backend upload)
- MySQL (optional)

## Cara Pakai

1. Clone repo ke `htdocs/` (XAMPP)
2. Akses `http://localhost/Portofolio/`
3. Admin panel: `http://localhost/Portofolio/admin/`
4. Password admin: `admin123`
